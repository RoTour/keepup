# Authoring

A Trainer writes open-ended questions ahead of class and states, for each one, what a good answer must contain. Nothing in this context knows that a class will ever happen.

## Language

**Trainer**:
The person who writes a Quiz. Here they are an author, and nothing more.
_Avoid_: Intervenant, Formateur, Instructor, Teacher, Speaker

**Quiz**:
A set of open-ended questions, each carrying the criteria its answers will be graded against. Written once, reusable across many Sessions.
_Avoid_: Test, Survey, Questionnaire, Assessment

**Question**:
An open-ended prompt inside a Quiz. It has no model answer — only Criteria.
_Avoid_: Item, Prompt, Exercise

**Criterion**:
One statement, written by the Trainer, of something a good answer must contain. A Question is graded against its Criteria and nothing else.
_Avoid_: Rubric, Rule, Expected answer, Correct answer
