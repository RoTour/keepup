# Authoring

A Trainer writes open-ended questions ahead of class and states, for each one, what a good answer must contain. Nothing in this context knows that a class will ever happen.

## Language

**Trainer**:
The person who writes a Quiz. Here they are an author, and nothing more.
_Avoid_: Intervenant, Formateur, Instructor, Teacher, Speaker

**Quiz**:
A set of open-ended questions, each carrying the criteria its answers will be graded against. Written once, reusable across many Sessions.
_Avoid_: Test, Survey, Questionnaire, Assessment

**Question**:
An open-ended prompt inside a Quiz. It has no model answer — only Criteria.
_Avoid_: Item, Prompt, Exercise

**Criterion**:
One statement, written by the Trainer, of something a good answer must contain. A Question is graded against its Criteria and nothing else.
_Avoid_: Rubric, Rule, Expected answer, Correct answer

**Collection**:
The Quizzes a Trainer owns — the ones they wrote and the ones they Imported. Private to that Trainer, except for what they have Shared.
_Avoid_: Library, Workspace, My quizzes

**Share**:
The owner's act of making a Quiz Public, reversible at any time. Un-sharing hides it from other Trainers; copies already Imported are untouched.
_Avoid_: Publish (the queue publishes messages), Post, Release (a Delivery act)

**Public**:
The state of a Shared Quiz: every Trainer can see it and Import it. Off by default. Public exposes the Quiz's content — questions and criteria — never its owner's Sessions.
_Avoid_: Published, Open, Community

**Import**:
A Trainer's act of copying a Public Quiz into their own Collection. The copy is theirs entirely — editable, shareable, runnable — with no link back beyond lineage.
_Avoid_: Clone, Fork, Copy, Subscribe
